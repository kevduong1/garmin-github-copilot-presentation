MCPs & plugins

MCP is a common way to let an agent use external systems safely and consistently.

A plugin is a distributable bundle—often combining skills, MCP connections, and workflow defaults.

Garmin MCP registryA discoverable catalog of approved connections
MCP serverExposes tools and data from a system such as GitHub, a build service, or internal docs
PluginPackages capabilities into an easy-to-install experience for a team
Agent → selected tool → governed system access
