Skills

Common instructions/flows for an agent that gets loaded based on the context

skill directory

A skill is a folder with one required entry point.

skills/ └── read-requirements/ ├── SKILL.md ← required ├── get_requirements.py └── references/ └── requirement-format.md

SKILL.md tells the agent when to use the skill and how to run its workflow. Scripts and references are optional supporting resources.

skills / read-requirements / SKILL.md
Front matter
---
name: read-requirements
description: Use this skill whenever you encounter
  requirement tags like R12345678 or a user asks
  to read or review requirements.
---
Skill instructions

Read Requirements

## Command

Run get_requirements.py, located next to this SKILL.md, with one or more requirement IDs separated by spaces.

python <skill-path>\get_requirements.py `
