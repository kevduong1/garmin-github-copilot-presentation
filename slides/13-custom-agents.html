Custom agents

A custom agent is a named, preconfigured collaborator with a defined role, instructions, and tool access.

A custom agent

Owns a role

Shapes the agent’s identity, operating rules, and available tools for a recurring job.

EXAMPLE: “Firmware review agent”
A skill

Teaches a task

Adds focused, reusable expertise that any appropriate agent can load when the task calls for it.

EXAMPLE: “Run the release checklist”
